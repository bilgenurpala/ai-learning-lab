## 📅 Progress Log

### Day 1 – Working with Variables

**📌 Topics Covered**
- `print()` function
- Variables & dynamic typing
- Variable naming rules
- Basic data types (`str`, `int`)
- f-strings
- `input()` function
- Type conversion (`int()`)

**🧠 Key Concepts**

| Concept | Description |
|---|---|
| Variable | A container used to store data |
| Dynamic Typing | Python infers types automatically at runtime |
| f-String | Modern string formatting available since Python 3.6+ |
| Type Casting | Converting types explicitly, e.g. `int(input(...))` |

**💻 Practice**
- Variable creation and usage
- Printing formatted sentences with f-strings
- Collecting and converting user input
- Applying clean `snake_case` naming conventions
